## Featured Project

### Azure Insurance Data Engineering Pipeline

**Repository:**
https://github.com/Samirsukhwaba/Azure-Insurance-Data-Engineering

This project demonstrates an end-to-end Azure Data Engineering solution built using Microsoft Azure services.

Key features include:

* Built end-to-end ETL/ELT pipelines using Azure Data Factory
* Implemented a Medallion Architecture with Bronze, Silver, and Gold layers
* Performed large-scale data transformation using PySpark in Azure Databricks
* Stored curated datasets in Delta Lake
* Connected on-premises SQL Server using Self-Hosted Integration Runtime (SHIR)
* Developed reusable metadata-driven pipelines with dynamic parameters
* Automated pipeline execution with Azure Data Factory triggers
* Secured secrets using Azure Key Vault
* Implemented CI/CD using Azure DevOps and ARM Templates
* Worked with CSV, Parquet, and Delta file formats
* Added monitoring, validation, and error handling to improve pipeline reliability
* Improved pipeline performance by approximately 30% through Delta Lake optimization, partitioning, and caching
